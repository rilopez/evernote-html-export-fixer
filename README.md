# evernote-html-export-fixer

Converts html files generated by evernote's multi-page html  export to PDF


## installation 

```bash
 #requirements
sudo apt install pandoc # this program uses pandoc to convert HTML to PDF with latex
sudo apt-get install texlive-latex-base texlive-fonts-recommended texlive-fonts-extra texlive-latex-extra
sudo apt-get install librsvg2-bin
sudo apt install libimage-exiftool-perl  # used to change pdf properties



npm install -g .
```


## Usage

```
evernote-html-to-pdf 
Usage: -n <name>

Options:
      --help       Show help                                           [boolean]
      --version    Show version number                                 [boolean]
  -d, --dir        folder with  evernote multi-page html exports
                                                             [string] [required]
  -m, --maxfiles   Maximum number of files to process in directory, process all
                   if zero                                 [number] [default: 0]
  -p, --absprefix  abs directory prefix,  usually is the destiation path used
                   during evernote export.               [string] [default: "Z:"]


```

## Examples
```bash
evernote-html-to-pdf -d "path/evernote-exported/html-files"
```
